# java-filmorate
Template repository for Filmorate project.

# Database Diagram
![diagram](DatabaseDiagram.png)

## Basic entities:
* ### User
* ### Film

## Film contains entities:
* ### MPA (Motion Picture Association)
* ### Genre

## The possible values of MPA:
* ### G - No age restrictions for the film
* ### PG - Parental guidance is suggested for children
* ### PG-13 - The film may be inappropriate for children under 13
* ### R - Viewers under 17 require an accompanying or adult guardian
* ### NC-17 - No one under 18 is admitted
